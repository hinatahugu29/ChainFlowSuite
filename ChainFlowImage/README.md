# ChainFlow Image Tool 🖼️⚡✨

**ChainFlow Image Tool** は、大量の画像に対する一括リサイズや形式変換を爆速で処理する、パフォーマンス重視のプロフェッショナル・ユーティリティです。

---

## 🚀 パラレル・フォージ / Core Features

### ⚡ Multi-threaded Processing
`ThreadPoolExecutor` を活用し、CPU コアの限界まで引き出した並列処理により、数千枚の画像も一瞬で完了。大量の素材を扱うワークフローにおいて、待機時間をゼロにします。

### 🪄 Smart Format Conversion
WEBP, PNG, JPG への相互変換。JPEG 変換時の透過背景の自動白埋め処理など、実務で必要とされる細やかな配慮が自動で実行されます。

### 📐 Independant Resize Modes
指定ピクセルへの幅固定、またはパーセント指定による比率維持リサイズを強力にサポート。用途（Web 用、SNS 用、印刷原稿用）に合わせた最適な出力を実現します。

---

## 🎨 ワークフローの効率化 / Efficiency

- **Drag & Drop Flow**: Filer やエクスプローラーから直接画像をドロップし、変換設定を選んで実行するだけの極めてシンプルなフロー。
- **Processed Logging**: 各ファイルの結果（成功・失敗・出力先）をリアルタイムにログ表示。大量処理の安心感を担保します。
- **Suite Identity**: 他の ChainFlow ツールと同等の深淵なダークテーマを適用。長時間の作業でも一貫した体験を提供します。

---

## ⌨️ 主要ショートカット / Shortcuts
- **`Delete`**: リストから選択項目を削除。
- **`Esc`**: 処理の中断。
- **`Enter`**: 処理の開始。

---

## 🛠️ 技術スタック / Tech Stack
- **Framework**: PySide6 (Qt for Python 6)
- **Processing Core**: PIL (Pillow) による高品質・高速エンコード
- **Threading**: 非同期 GUI アーキテクチャ
