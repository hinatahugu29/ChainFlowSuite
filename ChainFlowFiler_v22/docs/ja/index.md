# Chain Flow Filer v20.1 ドキュメント

**Chain Flow Filer** へようこそ。
これは、キーボード操作主体のパワーユーザー向けWindows用ファイルマネージャーです。
「深く潜る」カラム表示と「広く並べる」マルチウィンドウの利点を融合し、かつてない没入感と効率を提供します。

## v20.1 新機能

### 🖊️ 統合Markdownエディタ
ファイラから直接Markdownファイルを編集できるようになりました。
*   **リアルタイムプレビュー**: ダークテーマで見やすい表示
*   **PDF出力**: 白背景のプロフェッショナルな書類スタイルで出力
*   **Slash Menu (Ctrl+P)**: コマンドとMarkdownスニペットへのクイックアクセス

### 🌙 ダークテーマ完全統一
*   タイトルバー、メッセージボックス、動的生成ペインを含む全UIがダークテーマに
*   Windows DWM APIを活用したネイティブな見た目

### 🔍 Chain Flow Search (v20.1)
*   **タブ対応検索**: 複数の検索をタブで切り替えられる軽量検索ツール
*   **Filer連携**: カレントフォルダから即座に検索を開始

### 🛠️ UI/UX ポリッシュ (v20.1)
*   **PDF Merger**: ダークタイトルバー適用
*   **Bug Fixes**: Alt+Tab時のハイライト残留、PDF変換後のカーソルスタックを修正

## 主要機能

*   **Chain Flow Interface**: 親→子→孫へと連なるフローを視覚的に管理。
*   **Keyboard Centric**: ほぼ全ての操作をキーボードだけで完結。
*   **Quick Look**: Spaceキーで瞬時にファイルをプレビュー。
*   **Context Highlight (Alt)**: 祖先・子孫フォルダをハイライト表示。
*   **Hover Auto-Focus**: マウスを動かすだけで直感的にフォーカス移動。
*   **Unified Batch Drag**: 異なるフォルダのファイルを一括収集してドラッグ＆ドロップ。
