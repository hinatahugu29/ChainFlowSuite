# ChainFlowWriter v4 究極の機能・表現能力カタログ

ChainFlowWriter v4 は、標準的な Markdown エディタの枠を超え、**「事務報告書の作成を極限まで効率化・高品位化する」**ために最適化された執筆環境です。
このドキュメントでは、現在利用可能なすべての主要機能を網羅的に解説します。

---

## 1. コア・ライティング機能（基本の表現力）

標準的な Markdown に加え、実務に必要な高度なレンダリングエンジンを搭載しています。

- **標準 Markdown**: 見出し、リスト、テーブル、リンクに加え、**打ち消し線**（`~~`）もサポート。
- **LaTeX 数式 (KaTeX)**: `$$` で囲むブロック数式や `$ $` のインライン数式を、論文品質で美しく描画。
- **Mermaid ダイアグラム**: ガントチャート、フローチャート、シーケンス図などをコードから生成。
- **シンタックスハイライト (highlight.js)**: コードブロックを VS2015 系の落ち着いたダークテーマで強調表示。

---

## 2. 動的コンテンツ＆自動化（効率化の武器）

「一度書いたものを再利用する」「定型作業を自動化する」ための機能群です。

- **Front Matter 変数置換**:
  ドキュメント冒頭の YAML 形式で変数を定義し、本文中の `{{key}}` で呼び出せます。
  > 案件名や日付を一箇所で管理し、ミスなく文書全体に反映。
- **Macros メニュー**:
  - **日付テンプレート**: 日本語形式、海外形式（December...）、ISO形式をワンクリック挿入。
  - **変数雛形**: 置換機能のサンプルを即座に呼び出し。
- **電子印影・スタンプ機能**:
  `{stamp:承認}` のような記法で、本格的な電子印影（丸印）を生成。

---

## 3. 次世代レイアウト＆デザイン（v4 の目玉機能）

HTML の柔軟性と Markdown の手軽さを融合させた、本アプリ独自の強力な機能です。

- **魔法のタグ `<m-d>` (Markdown in HTML)**:
  HTML の `<div>` や `<table>` の中でも Markdown が使えます。
  - **自動デデント**: HTML のネストに合わせて字下げしていても、中身を賢く解析。
  - **インライン最適化**: 不要な `<p>` タグを生成せず、タイトなレイアウトを実現。
- **コードブロック背景色の動的カスタマイズ (`code_bg`)**:
  フロントマターに `code_bg: "#カラーコード"` と書くだけで、コードブロックの背景色を変更。
- **インタラクティブ・コピー・スニペット**:
  プレビュー画面をクリックするだけで、定型文をクリップボードにコピー可能。

---

## 4. プロフェッショナル出力 (PDF Fidelity)

「プレビューで見たものが、そのまま PDF になる」ことを徹底追求しています。

- **WYSIWYG PDF Export**:
  JavaScript による動的なレイアウト計算結果を物理的に PDF エンジンへ強制継承。
- **スマート・ページブレイカー**:
  - **ページガイド表示**: 印刷時の境界線をリアルタイムでプレビュー。
  - **自動余白補正**: 改ページ直後のマージン消失を防止し、常に一定の余白を保持。
- **Refresh Preview (Final Check)**:
  エクスポート直前にレイアウトを再計算するための専用ボタン。

## 5. 機能・挙動リファレンス表

| カテゴリ | 機能名 | 指定方法 / 記法 | 特徴 |
| :--- | :--- | :--- | :--- |
| **自動置換** | 変数埋め込み | `{{key}}` | 属性、Mermaid、LaTeX内でも有効 |

## 🎯 迷った時の使い分けガイド

1. **「案件情報を一発で全置換したい」**
   → 文末の **「Front Matter 変数」** を使いましょう。
2. **「複雑なカード型レイアウトの中にリストを作りたい」**
   → HTML タグの内側に **`<m-d>`** を忍ばせましょう。
3. **「報告書の雰囲気をガラッと変えたい」**
   → フロントマターの **`code_bg`** やフォント設定を調整しましょう。
4. **「PDF で変なところで改ページされたくない」**
   → サイドバーの **「Show Page Guides」** を見ながら執筆しましょう。
