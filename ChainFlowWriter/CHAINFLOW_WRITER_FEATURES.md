# ChainFlow Writer v21
## 「書く」を加速させ、美しく形にする。次世代 Markdown ライター。

ChainFlow Writer は、単なるテキストエディタではありません。思考を途切れさせることなく、プロフェッショナルな品質のドキュメントを最短距離で出力するために設計された、**高機能 Markdown ドキュメンテーション・ツール**です。

---

### 💎 圧倒的な「売り」と魅力

#### 1. 究極のライブプレビュー・エクスペリエンス
*   **「印刷イメージ」を常時確認**: 単なるHTMLプレビューではなく、余白（マージン）、ページ区切り、ヘッダー/フッターをリアルタイムに計算。PDF出力後の「ズレ」に悩まされることはもうありません。
*   **プロフェッショナル・タイポグラフィ**: 遊明朝や Noto Serif JP を活用した、日本語が最も美しく見えるレイアウトを追求。

#### 2. 表現力を解き放つ拡張 Markdown
*   **数式・図解の完全対応**: KaTeX による美麗な数式レンダリング、Mermaid によるフローチャートやガントチャートの記述がシームレスに行えます。
*   **カスタム・コールアウト**: `info`, `warning`, `alert` といった装飾ブロックを標準装備。エンジニアリングドキュメントやマニュアル作成に最適です。
*   **「電子印影」機能**: 独自の `stamp` 記法により、ドキュメントに実印や承認印のようなビジュアルを重ねることが可能。

#### 3. 爆速の執筆を支える「スニペット・エンジン」
*   **独自のライブラリ構築**: よく使うレイアウト、複雑な表組み、定型文をスニペットとして登録。
*   **サムネイル付きカタログ**: 登録したスニペットはビジュアル付きの「チートシート」から一目で選択・挿入可能。あなたのナレッジが、そのまま執筆の武器になります。

#### 4. 高度な PDF エクスポート・エンジン
*   **ピクセル・パーフェクト**: プレビューで見たままの品質で PDF を生成。
*   **フロントマター制御**: YAML 形式のメタデータで、フォントサイズ、マージン、用紙サイズをドキュメントごとに自在にコントロール。

#### 5. 洗練された「ChainFlow」デザイン
*   **Modern Dark UI**: VSCode のような集中力を高めるダークテーマを採用。
*   **エンジニア・フレンドリー**: 軽量・高速動作。ポータブル設計により、どこでも自分の環境が再現可能。
