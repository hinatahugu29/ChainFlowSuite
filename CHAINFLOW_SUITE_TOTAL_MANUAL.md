## 0. イントロダクション

### 0.1 ChainFlow Suite とは
ChainFlow Suite は、Windows環境における事務ワークフローを極限まで効率化するために設計された、プロフェッショナル向けツール群のコレクションです。個別のツールは独立して動作しつつも、共通の設計思想（キーボード主体、ダークテーマ、高速レスポンス）と、統一されたブランドアイデンティティにより、シームレスな体験を提供します。

### 0.2 コア・コンセプト：Hybrid Flow（ハイブリッドな情報の連鎖）
情報の「Flow（連鎖）」は、以下の4つのフェーズを通じて循環します。

1. **収穫・抽出 (Harvest/Extract)**: Webや各種資料から「価値ある資産」を純度100%で射抜く。
2. **探索・整理 (Explore/Organize)**: 膨大なデータから「瞬時」かつ「確実」に目的のファイルへ到達する。
3. **思考・構築 (Think/Build)**: MarkdownとHTML/CSSを融合させ、構造化された美しいドキュメントや図解を作成する。
4. **完成・提出 (Finish/Submit)**: 固定された成果物（PDF）として結晶化し、比較、加工、検証を行う。

### 0.3 技術的アイデンティティ
- **Intelligence (Python)**: 高度なビジネスロジックとAI連携の指令塔。
- **Force Core (Rust)**: 物理層に近いレイヤーでの並列処理、高速 I/O。
- **Stability Guard (shiboken6)**: C++ オブジェクトの生存管理による、ハイブリッド構成での「絶対的安定」の担保。

## 1. セットアップと共通基盤

### 1.1 動作環境
- **OS**: Windows 10 / 11 (64bit 推奨)
- **Runtime**: Python 3.10 以上
- **主要依存ライブラリ**: PySide6 (Qt for Python), PyMuPDF, markdown-it-py, (一部で Rust コンパイル済みバイナリを使用)

### 1.2 導入手順
1. リポジトリをクローンまたはダウンロードします。
2. `py -m pip install -r requirements.txt` を実行し、必要な依存関係を解決します。
3. 各ツールのディレクトリにある `main.py` を `py` コマンドで実行します。
   - 例: `py ChainFlowFiler_v23_2\main.py`

### 1.3 共通の操作作法
- **Slash Menu (`Ctrl+P`)**: 全ての共通操作とツール起動のハブ。
- **ダークモード標準**: 長時間の事務作業でも疲労を最小限に抑える配色。
- **AppUserModelID 統合**: 全てのツールがタスクバー上で一つの「ChainFlow Suite」として統合され、切り替えが容易。

## 2. コンポーネント別詳細リファレンス

### 📁 ChainFlow Filer v23.2 (The Modular Control Tower)
スイート全域を統括する「情報の司令塔」です。

- **実行ファイル**: `ChainFlowFiler_v23_2/main.py`
- **特徴**:
    - **Rustネイティブコア**: フォルダ走破とファイル走査を Rust で超高速化。
    - **I/O負荷ゼロ描画**: アイコン生成をインメモリで行い、ディスクI/Oを極限まで削減。
    - **Native Sort Engine**: 数万件のファイルも一瞬で並び替える Rust 製ソートエンジン。
    - **Stability Guard**: `shiboken` によるメモリ安全管理で、高速操作時のクラッシュを根絶。
- **主要な使い方**:
    - **瞬時のフィルタリング**: フォルダ入力欄や検索バーへの入力で、表示ファイルをリアルタイムに絞り込み。
    - **Slash Menu (`Ctrl+P`)**: 他の Suite ツール（Writer, Sniper, Search 等）を現在開いているパスを引き継いだ状態で起動。
    - **マークシステム**: 重要なファイルにマーク（強調）を付け、視覚的に管理。

### ✍️ ChainFlow Writer v9 (The Document Engine)
IDE級の視認性とDTPの精度を兼ね備えた構造化ドキュメント作成ツールです。

- **実行ファイル**: `ChainFlowWriter_v9/main.py`
- **特徴**:
    - **リアルタイム改ページ**: 編集画面がそのままPDFの完成図としてレンダリングされる、独自のレイアウトエンジン。
    - **Stamp 構文**: `<stamp>` タグにより、印影や署名をミリ単位の絶対座標で配置可能。
    - **Snippet Manager (Cheat Sheet)**: コールアウト、2カラム、画像キャプション等の再利用可能なコンポーネントを瞬時に挿入。
    - **YAML Frontmatter**: 余白、タイポグラフィ、ページサイズ等のレイアウト設定をドキュメント内に保持。
- **主要な使い方**:
    - **ドキュメント執筆**: Markdown形式で執筆。`Ctrl+Alt+B` でチートシートを表示し、よく使う構造を挿入。
    - **レイアウト調整**: 設定パネル (`Ctrl+R`) で余白やフォントをリアルタイムに変更。
    - **PDF出力**: `Ctrl+P` で、プレビュー通りの1:1の再現性を持つ高品質PDFを生成。

### 🔍 ChainFlow Sniper V3 (The Research Shell)
Web上の「構造」と「美」を正確に射抜く、プロフェッショナル向けリサーチブラウザです。

- **実行ファイル**: `ChainFlowSniper_V3/main.py`
- **特徴**:
    - **4象限抽出（Sniper Modes）**: A (Text), Z (Markdown), S (Style), D (Pure DOM) の各モードで目的の形式に合わせて情報を抽出。
    - **深層貫通探索**: iFrame、Shadow DOM、透明レイヤーを透過して値を抽出。
    - **共有セッション (Force Persistent)**: タブ間、セッション間でログイン状態を共有。
- **主要な使い方**:
    - **スナイプ操作**: 目的のキー（A/Z/S/D）を長押ししながら要素をクリック。ホイール上下で階層を調整。
    - **バッチ・スナイプ**: `C` キー長押しで複数の項目を連続クリックし、離した瞬間に一括抽出。
    - **デタッチ (Detach)**: 抽出リストを別窓で切り出し、外部エディタの隣に配置してリサーチを効率化。

### 🎨 ChainFlow Designer (The Layout Master)
直感的な操作とインテリジェンスが融合したDTPエディタです。

- **実行ファイル**: `ChainFlowDesigner/main.py`
- **特徴**:
    - **Smart Snapping**: オブジェクトの端点や中心、他の要素との整合性を検知して吸着。
    - **Pro Table Widget**: Excelライクな操作で表のセル幅や高さを個別にリサイズ。
    - **完全な Undo/Redo**: すべての操作を履歴管理。
- **主要な使い方**:
    - **レイアウト設計**: アイテムを配置し、`Ctrl+G` でグループ化、プロパティパネルで微調整。
    - **出力**: 300DPI相当の高解像度画像や、ガイド込のPDFとしてエクスポート。

### 🔍 ChainFlow Search v23 (The Information Oracle)
超高速スキャンとインテリジェンスな絞り込みを統合した検索エンジンです。

- **実行ファイル**: `ChainFlowSearch_v23/main_v23.py`
- **特徴**:
    - **並列スキャンエンジン**: マルチコアをフル活用し、ファイルシステムを物理限界に近い速度で走破。
    - **Advanced Query**: Space(AND), `|`(OR), `-`(NOT) などの高度なクエリ記法。
- **主要な使い方**:
    - **マルチタブ検索**: `Ctrl+T` で複数の検索を並行実行。
    - **履歴連携**: 過去の検索ディレクトリに瞬時にアクセス。

## 3. 実践的な運用ワークフロー

### 3.1 リサーチから報告書作成への連鎖（Flow）
1. **[Sniper]**: ブラウザで調査を行い、必要なデータやスタイルをスナイプ。リストとして蓄積。
2. **[Filer]**: スナイプしたデータの保存先ディレクトリへ移動。`Ctrl+P` で Writer を起動。
3. **[Writer]**: Sniper の抽出リストを基に報告書を執筆。チートシートから定型枠を挿入。
4. **[Designer]**: 複雑な図解が必要な場合、Filer から Designer を起動。図を作成して画像出力。
5. **[Writer]**: 出力した図を読み込み、`Ctrl+P` で最終的なPDFとして書き出し。

## 4. テクニカルリファレンス

### 4.1 ビルドと配布
- **ビルドスクリプト**: `build_v22.bat` を使用することで、全てのアプリを単体実行可能な EXE 形式へ一括コンパイルします。
- **Nuitka / PyInstaller**: 内部では C への変換（Nuitka）とパッケージング（PyInstaller）を最適に組み合わせています。

### 4.2 安定性とトラブルシューティング
- **ログ管理**: クラッシュや予期せぬ動作が発生した場合は、各ディレクトリの `*.log` ファイル、または標準出力から shiboken6 の警告を確認してください。
- **メモリ安全**: 大規模なファイル操作中にフリーズが発生する場合、Rust コアと Python 間のデータ受け渡しバッファを確認する必要があります。
