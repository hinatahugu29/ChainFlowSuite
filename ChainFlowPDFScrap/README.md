# ChainFlow PDF Scrap 📄✂️✨

**ChainFlow PDF Scrap** は、PDF 内部の特定のエリアや構造を瞬時に「切り抜き」、再利用可能な素材へと変換するための、実務特化型のスクラップステーションです。

---

## 🚀 抽出・スクラップ機能 / Key Features

- **Area Clipping**: PDF ページ内の任意の範囲をドラッグで選択し、高解像度画像として即座に抽出。
- **Intelligent Flow Integration**: 抽出したスクラップを直接 Filer や Writer、あるいは外部の Office アプリへドラッグ＆ドロップで流し込むことが可能です。
- **Structured Scrapping**: 表組みや図解など、後から参照したい重要な箇所を座標レベルで保持し、 Suite エコシステム内で共有。

---

## 🛠️ 技術スタック / Tech Stack
- **Framework**: PySide6 (Qt for Python 6)
- **Core Engine**: PyMuPDF による精密なレンダリングとキャプチャ
- **Interaction**: 直感的なマウス操作と高速プレビュー
