# ChainFlow ToDo 📝

**ChainFlow ToDo** は、美しさと操作性を妥協しない、パワーユーザーのためのミニマル・タスク管理ツールです。

### ✨ 主な機能 / Key Features
- **Category Management**: タスクを自由にグループ化。各カテゴリは開閉（アコーディオン）可能で、必要な情報のみを集中して表示。
- **Auto Progress Visualization**: カテゴリごとの進捗率をリアルタイムに計算し、ヘッダーの進捗バーに反映。
- **Markdown Export**: 全てのタスク、チェック状況、タイムスタンプを階層化されたMarkdown形式で一括コピー。Noteやドキュメントへの転記が容易。
- **Priority Badging**: 高・中・低の優先度を色彩的に管理。一目で重要度を識別可能。
- **Animated UI**: 滑らかな開閉アニメーションと、ホバーエフェクトによるモダンなユーザー体験。

### 🛠️ 技術情報 / Technical Info
- **Framework**: PySide6
- **Persistence**: JSONベースの軽量データストレージ。
- **Theming**: Suite統一の深淵なダークテーマを適用。

### ⌨️ 操作方法 / Shortcuts
- `Enter`: 新しいタスクの追加
- `Space`: チェックボックスの切り替え
- `Delete`: 項目/カテゴリの削除（確認ダイアログ付き）
- `Ctrl + C`: Markdown形式で全体をエクスポート
