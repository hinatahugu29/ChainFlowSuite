# ChainFlow ToDo

[Japanese] | [English]

## 🌟 Overview / 概要

**ChainFlow ToDo** is a minimalist, keyboard-centric productivity tracker designed to stay out of your way while keeping your tasks organized. It integrates perfectly into the ChainFlow Suite ecosystem.

**ChainFlow ToDo** は、作業の邪魔をせず、かつタスクを確実かつスマートに管理するために設計されたミニマルな ToDo 管理ツールです。ChainFlow Suite エコシステムの一部として、シンプルながら強力なワークフローを提供します。

---

## ✨ Key Features / 特徴

*   **Keyboard-First**: Quickly add, edit, and complete tasks with shortcuts.
*   **Clean UI**: Dark-mode optimized, clutter-free interface.
*   **Suite Integration**: Can be launched instantly via Filer's Slash Menu (`Ctrl+P`).
*   **キーボード主体の操作**: ショートカットによりタスクを素早く追加・編集・完了。
*   **洗練されたUI**: ダークモードに最適化された、ノイズのないインターフェース。
*   **スイート連携**: ファイラーのスラッシュメニューから即座に起動可能。

---
