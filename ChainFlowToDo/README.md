# ChainFlow ToDo 📝✨

**ChainFlow ToDo** は、美しさと操作性を極限まで追求した、パワーユーザーのためのミニマル・タスク管理ハブです。
「思考を妨げない」ことを最優先に設計された UI と、Suite 全体との高い親和性により、実務のフローを淀みなく管理します。

---

## 🚀 思考を加速させるタスク管理 / Key Features

### 📂 Dynamic Category Management
タスクを自由にグループ化し、アコーディオン形式で開閉可能。必要な情報だけに集中できる「ノイズレス」な環境を提供します。

### 📊 Real-time Progress Visualization
カテゴリごとの進捗率をリアルタイムに計算し、流麗なアニメーションを伴うプログレスバーに反映。全体の達成度を一目で把握できます。

### 📑 Markdown Flow Integration
すべてのタスク、完了状況、タイムスタンプを階層化された Markdown 形式で一括エクスポート。Writer や Note、報告書への転記が瞬時に完了します。

---

## 🛠️ 洗練されたユーザーインターフェース / UI & UX

- **Priority Badging**: 優先度（高・中・低）を色彩心理に基づいて管理。視覚的に重要度を識別。
- **Animated Interactions**: 滑らかな開閉アニメーションと、ホバーエフェクトによるモダンで「手馴染み」の良い操作感。
- **Suite Theming**: 深淵なミッドナイトブルーを基調とした、プロフェッショナル仕様のダークテーマ。

---

## ⌨️ 主要ショートカット / Shortcuts
- **`Enter`**: 新しいタスクの追加。
- **`Space`**: チェックボックスの切り替え。
- **`Delete`**: 項目またはカテゴリの削除。
- **`Ctrl + C`**: Markdown 形式でエクスポート。

---

## 🛠️ 技術スタック / Tech Stack
- **Framework**: PySide6 (Qt for Python 6)
- **Persistence**: 軽量な JSON ベースのデータストレージ
- **Identity**: AppUserModelID による Suite 統合
