# ChainFlow PDF Studio

[Japanese] | [English]

## 🌟 Overview / 概要

**ChainFlow PDF Studio** is a specialized tool for PDF manipulation and analysis within the ChainFlow Suite. It provides advanced features for handling PDF documents that standard viewers lack.

**ChainFlow PDF Studio** は、ChainFlow Suite において PDF 文書の操作と解析に特化した専門ツールです。標準的なビューワーでは不可能な、高度な PDF 処理機能を提供します。

---

## ✨ Key Features / 特徴

*   **Advanced Viewing**: Optimized for technical documents and layouts.
*   **Suite Synergy**: Deeply integrated with ChainFlow Writer and Filer.
*   **Toolbox**: Specialized functions for PDF metadata and structure analysis.
*   **高度なビューワー**: 技術文書や複雑なレイアウトの確認に最適化。
*   **スイート・シナジー**: ChainFlow Writer や Filer との深い連携。
*   **ツールボックス**: PDF のメタデータや構造解析のための専門機能。

---

## 🚀 How to Run / 実行方法

```bash
python main.py
```
*(Requires PySide6, PyMuPDF or equivalent)*
