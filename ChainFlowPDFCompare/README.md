# ChainFlow PDF Compare 📄⚖️✨

**ChainFlow PDF Compare** は、二つ以上のドキュメント間の微細な差異を逃さない、厳密な校正と検証のためのプロフェッショナル比較ステーションです。

---

## 🚀 知的な比較テクノロジー / Precision Proofing

### 🛡️ Multi-View Sync Engine
1 画面から最大 4 画面まで、用途に合わせたレイアウト構成を瞬時に切り替え（1, 2H, 2V, 4 Views）。すべてのビューを完全に同期してスクロールさせ、異なるバージョンの資料を同一箇所で精密に比較・検証可能です。

### 🧘 ZEN Mode (F11)
UI ツールバーやサイドバーをすべて隠し、広大な画面全体を「比較作業」のみに捧げる全画面集中モード。思考のノイズを完全に排除し、文書の差異のみに意識を集中させます。

### 🖥️ Pop-out Workspace
特定のドキュメントを別ウィンドウとして独立させ、マルチディスプレイ環境を最大限に活用。大規模な資料比較における生産性を劇的に向上させます。

---

## ⌨️ 主要ショートカット / Shortcuts
- **`F11`**: ZEN モード（全画面集中）。
- **`1` / `2` / `4`**: 画面分割数の切り替え。
- **`S`**: 同期スクロールの ON / OFF。
- **`Delete`**: アクティブなビューの PDF をクリア。

---

## 🛠️ 技術スタック / Tech Stack
- **Framework**: PySide6 (Qt for Python 6)
- **Core Engine**: PyMuPDF (Fitz) による高速・高精細なベクトルレンダリング
- **Logic**: ビュー間の状態同期を司るリアクティブな UI 同期エンジン
