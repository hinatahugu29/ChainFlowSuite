# ChainFlow PDF Compare

[Japanese] | [English]

## 🌟 Overview / 概要

**ChainFlow PDF Compare** is a precision utility designed to detect even the slightest differences between two PDF documents. It is essential for ensuring fidelity after document updates or refactoring.

**ChainFlow PDF Compare** は、2つの PDF 文書間のわずかな差異をも検出するために設計された精密ユーティリティです。ドキュメントの更新やリファクタリング後の完全な再現性を保証するために不可欠なツールです。

---

## ✨ Key Features / 特徴

*   **Visual Diff**: Highlight added, removed, or modified elements between two versions.
*   **Fidelity Assurance**: Perfect for verifying ChainFlow Writer's PDF output stability.
*   **High Performance**: Efficiently compares large PDF files side-by-side.
*   **ビジュアル比較**: 2つのバージョン間で、追加・削除・修正された要素をハイライト表示。
*   **再現性の保証**: ChainFlow Writer で出力した PDF の整合性検証に最適。
*   **高パフォーマンス**: 大容量の PDF ファイル同士も、並べて効率的に比較。
