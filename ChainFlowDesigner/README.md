# ChainFlow Designer 🎨✨

**ChainFlow Designer** は、執筆内容に視覚的な「構造」と「説得力」を与えるための、プロフェッショナル向け DTP・レイアウトエディタです。
直感的な操作性とエンジニアリング級の精度を融合させ、事務実務における図解作成を極限まで効率化します。

---

## 🚀 知的なレイアウト・テクノロジー / Core Features

### 🧲 Smart Snapping (幾何学吸着)
オブジェクトの端点、中心点、あるいは他のオブジェクトとの水平・垂直方向の整合性を自動検知。磁石のように吸着するガイド機能により、誰でも「黄金比」を維持した美しい配置が可能です。

### 📊 Pro Table Architecture (Excel-Style Control)
表形式のオブジェクトにおいて、各行の高さや列の幅をドラッグで個別にリサイズ可能。コンテキストメニューから行・列の挿入や削除も自在に行え、複雑な表組みもエディタ感覚で構築できます。

### 💎 Advanced Export & Fidelity
用途に合わせて DPI（解像度）を指定可能な画像出力や、グリッドを含めた精密な PDF 出力をサポート。作成した図解を Writer 等の他 Suite アプリへ最高品質で提供します。

---

## 🛠️ プロ仕様の編集機能 / Advanced Specs

- **Unified Undo Stack**: すべてのアクションを完全に元に戻すことが可能な、堅牢なコマンドパターン基盤。
- **Layer & Swatches**: 複雑な重なりを管理するレイヤーパネルと、ブランドカラーを維持するスウォッチ管理。
- **Geometry Memory**: ウィンドウ位置やズーム倍率を保持し、再開時の「思考の断絶」を最小限に。

---

## ⌨️ 主要ショートカット / Shortcuts
- **`Ctrl + G` / `Shift + G`**: グループ化 / 解除。
- **`Ctrl + L` / `Shift + L`**: ロック / 解除。
- **`Ctrl + Z` / `Y`**: 元に戻す / やり直し。
- **`F11`**: **ZEN モード（全画面）**。

---

## 🛠️ 技術スタック / Tech Stack
- **Framework**: PySide6 (Qt GraphicsView)
- **Command Pattern**: `QUndoStack` による履歴管理
- **Format**: XML ベースの独自保存形式による完全な再現性
