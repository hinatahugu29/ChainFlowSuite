# Sniper Research Shell v2 ― マルチタブ対応 ウォークスルー

## 🚀 実装された主要機能

### 1. マルチタブ管理 (SniperTabManager)
- **QTabWidget** ベースのインターフェースを採用。
- `+` ボタン、または `Ctrl+T` で新しいタブを開けます。
- `×` ボタン、または `Ctrl+W` でタブを閉じられます（最後の1つは閉じられないようガード）。
- タブの名前にはページのタイトルが自動的に反映（最大15文字で省略表示）されます。

### 2. ページ間のシームレスな同期
- **セッション共有**: すべてのタブが同一の `SniperProfile` を共有するため、ログイン済みのページはどのタブで開いてもログイン状態が維持されます。
- **JSブリッジ (QWebChannel)**: 新しいタブが開くたびに [SniperBridge](file:///e:/CODE/Antigravity/web-parts/core/bridge.py#4-18) が自動的に接続され、どのタブからも抽出（A/Z/Cキー）が可能です。
- **履歴とURLバー**: アクティブなタブを切り替えると、自動的に URL バーの内容と閲覧履歴の登録が追従します。

### 3. パフォーマンスと安定性の向上
- **JSブリッジのリトライロジック**: `qt` オブジェクトの注入遅延に備え、[sniper.js](file:///e:/CODE/Antigravity/web-parts/resources/sniper.js) 側で bridge が確立されるまで 100ms おきに再試行するロジックを導入。
- **タブ内リンクの処理**: `target="_blank"` などの新規ウィンドウ要求を、自動的に「新しいタブ」としてキャプッチするように [createWindow](file:///e:/CODE/Antigravity/web-parts/core/browser.py#12-20) を拡張。

## 🛠 新しい操作方法 (ショートカット)

| ショートカット | アクション |
| :--- | :--- |
| **Ctrl + T** | 新しい空のタブ（Google）を開く |
| **Ctrl + W** | 現在のタブを閉じる |
| **Ctrl + L** | URL バーにフォーカスして即座に入力 |
| **Ctrl +左右キー** | 表示中のページの「戻る」「進む」（ブラウザ標準） |
