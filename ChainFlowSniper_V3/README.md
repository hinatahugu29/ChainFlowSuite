# Sniper Research Shell

Sniper Research Shellは、日々のリサーチや情報収集を圧倒的に効率化するための「抽出・収集特化型」のカスタムブラウザステーションです。通常のブラウザとしての機能（社内システムや複雑なWebアプリの利用）を一切妨げることなく、必要な情報だけを「狙撃」して即座に整理・蓄積します。

## 1. 起動方法
```powershell
# プロジェクトフォルダにて
py main.py
```

## 2. 画面構成 (3ペイン + 別ウィンドウ)
- **左ペイン (Favorites & History)**: \
  お気に入り機能（+/- ボタンで現在ページを追加/削除）と、過去に訪れたページのタイムトラベル（履歴クリックで復帰）を提供します。
- **中央ペイン (Browser)**: \
  安全な永続プロファイル（ログイン状態を保持）を備えたメインブラウザです。社内システム等でもログアウトされません。
- **右ペイン (Extractions)**: \
  ブラウザから「スナイプ（抽出）」したテキストや構造化データが蓄積されます。
- **Detach（切り離し）ポップアップ**: \
  右ペインの `Detach` ボタンを押すと、抽出リストが常に最前面の別ウィンドウとして独立します。他のアプリ（ExcelやNotionなど）を開きながら、蓄積したデータを一つずつコピペする作業に最適です。

## 3. 究極の操作体系（Sniper Operations）

スナイパーシェルは、以下の高度な抽出機能を備えています。

| 機能名 | 概要 |
| :--- | :--- |
| **スマート・ランキング** | クリックした「範囲」から、見出し（ラベル）ではなく「データ」を自動選別して抽出。 |
| **深層貫通探索** | IframeやShadow DOM、透明なレイヤーを透過して隠れた値をサルベージ。 |
| **自動クリーンアップ** | 抽出時に混入した余計な空白や、操作キー（a/z）をインテリジェントに除去。 |

## 3. 究極の操作体系（Sniper Operations）

スナイパーシェルは、**特定のキーをホールド（押しっぱなし）している間だけ抽出モード**になります。
キーを押していない時は、完全に「普通のブラウザ」として動作するため、リンクのクリックや入力フォームの操作を邪魔しません。

### 🖱️ 直感的な抽出操作 (ホールド ＋ クリック)

| 操作キー (長押し) | モード名 | 機能 |
| :--- | :--- | :--- |
| **【A】キー** | ダイレクト・スナイパー | **(青枠)** テキストやセルなど、ホバーしている「単一要素」を正確に狙撃して抽出します。 |
| **【Z】キー** | ストラクチャ・スナイパー| **(紫枠)** 対象を含む「表(Table)」や「リスト(UL/OL)」の構造全体を自動判別し、Markdown形式で一気に抽出します。 |
| **【C】キー** | 連続バッチ・スナイパー | 押している間、複数の要素を連続クリックで蓄積し、キーを離した瞬間に「一つにまとめて」抽出します。 |

### 🔧 抽出範囲の微調整 (ズーム)
* **【A】または【Z】キー + マウスホイール上下**: \
  ホバー中の枠線を、親要素（外側）へ広げたり、子要素（内側）へ狭めたりできます。画面上部には現在のHTML階層（パンくずリスト）が表示されます。

### 📋 その他の便利機能
* **スマート・スコアリング抽出**: \
  単なるクリックだけでなく、IDパターンや要素の性質から「値」を自動判別して優先的に抜き出します（eValue等の業務システムで威力を発揮）。
* **深層貫通探索 (Deep Penetration)**: \
  IframeやShadow DOMの内部、さらには透明なカバー要素に隠されたデータも透過して射抜くことが可能です。
* **項目の展開/コピー**: \
  抽出リスト内のアイテムをクリックすると、全文がクリップボードにコピーされます。ダブルクリックで項目の省略/全文表示の切り替えが可能です。

## 4. 今後実装予定の機能
- **データエクスポートの自動化**: 右ペインに溜まったリストを、Markdown/CSVファイルとしてワンクリック保存。
- **AI要約機能**: 抽出したテキスト群をサイドバー上で即座にAIに整理・要約させる連携機能。
- **スクラップブック管理**: 抽出データと併せて「抽出時のスクリーンショット」と「元URL」を紐付けて保存する機能。
