# Sniper Research Shell v3 - 機能・特徴ガイド

「Sniper Research Shell」は、Web上の情報をプロフェッショナルなリサーチワークフローへと昇華させるために設計された、インテリジェントなWebリサーチ専用ブラウザです。

## 🎯 コアコンセプト
従来のリサーチ作業（コピー＆ペーストの繰り返し）を「スナイプ（狙い撃ち）」という直感的な操作に置き換え、思考を止めずに情報の抽出と集約を行うことを目的としています。

---

## ⚡ 強力な抽出モード (Sniper Mode)
任意の要素にマウスを合わせ、以下のキーを押しながらクリックするだけで情報が得られます。

- **[A] ダイレクト・スナイプ (Direct Mode)**: テキストや数値を即座に抽出。入力フォームの中身も正確に取得します。
- **[Z] ストラクチャ・スナイプ (Structure Mode)**: テーブルやリストを Markdown 形式に変換。
- **[C] バッチ・スナイプ (Batch Mode)**: 連続クリックで複数の情報を自由に蓄積し、キーを離した瞬間に一括で抜き出し。
- **[S] スタイル・スナイプ (Style Mode)**: 要素の見た目（HTML/CSS）をそのまま軽量コンポーネントとして抽出。A4紙面作成などに使えるよう、レイアウト崩れ防止ラッパーを通じた自動調整を行います。
- **[D] DOM・スナイプ (DOM Mode) 【NEW in V3】**: CSSやクラスといった全ての装飾情報や寸法属性を根こそぎパージし、`<p>` や `<table>` といった純粋なセマンティックHTML構造の骨格のみを抽出。LLMへのコンテキスト渡しや、Writerテーマとの同化に特化したモードです。
- **スマート・ランキング (Smart Ranking)**: 項目名（ラベル）に惑わされず、中身の「データ」をインテリジェントに優先抽出。
- **深層貫通 (Deep Penetration)**: Iframe、Shadow DOM、透明なカバーレイヤーを透過して値をサルベージ。

### ✨ インテリジェントなクリーンアップ
テキストエリアや入力フォームから抽出する際、操作キー（a/z/c/s）が文字として混入してしまっても、抽出の瞬間に自動的に検知・削除されます。

---

## 🌐 インテリジェント・ブラウザ機能
- **マルチタブ対応**: `Ctrl + T` で新規タブ、`Ctrl + W` でタブを閉じ、複数のリサーチを並行して行えます。
- **共有セッション (Force Persistent)**: 全てのタブでログイン状態が共有され、「セッションCookie」であっても保持するため、一度ログインすれば社内システム等でもログアウトされません。
- **万能URLバー (`Ctrl + L`)**: URL入力だけでなく、直接キーワードを入力すればGoogle検索へ自動的にリダイレクト。
- **ステルス・アクセス**: Firefox に擬装した User-Agent により、Googleログイン等のセキュリティチェックをスムーズに通過。

---

## 🎨 プロフェッショナルなインターフェース
- **プレミアム・ダークテーマ**: 長時間の作業でも目が疲れにくい、洗練された配色。
- **画面上部パンくずリスト**: マウスホイールによる階層移動時、現在どこを狙っているかが一目でわかるUI表示。
- **デタッチ (Detach) 機能**: 抽出リストを別窓で切り離し、他の作業ツール（メモ帳やExcel）の隣に置いて作業効率を最大化。
- **お気に入り・履歴管理**: 頻繁に訪れるサイトやすぐに戻りたいページを直感的に管理。

---

## ⌨️ ショートカットキー一覧

| キー | アクション |
| :--- | :--- |
| **Ctrl + T** | 新しいタブを開く |
| **Ctrl + W** | 現在のタブを閉じる |
| **Ctrl + L** | URLバーにフォーカス |
| **A (長押し)** | テキストをスナイプ（クリックで確定） |
| **Z (長押し)** | テーブル/構造をスナイプ（クリックで確定） |
| **C (長押し)** | 連続スナイプモード（クリックした分を蓄積） |
| **S (長押し)** | スタイル（インラインCSS）込みでコンポーネントをスナイプ |
| **D (長押し)** | スタイルを除外して純粋なHTML構造（DOM）のみをスナイプ |
| **上記のキー + マウスホイール上下** | スナイプ対象の要素範囲（階層）の拡大 / 縮小 |

## 🔒 安心・安全の設計
- **XSS保護**: 抽出したテキストやHTMLの取り扱い時、プレビュー等で悪意のあるスクリプトが実体化しないよう設計。
- **アトミック保存**: データの保存時に一時ファイルを使用し、不慮のクラッシュによるデータ消失を防ぎます。
- **完全ローカル**: 抽出したデータや履歴は、外部サーバーではなく全てローカルの `.web_data/` 内に安全に保存されます。
