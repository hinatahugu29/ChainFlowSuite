# ChainFlowWriter v6 究極の機能・表現能力カタログ

ChainFlowWriter v6 は、標準的な Markdown エディタの枠を超え、**「事務報告書の作成を極限まで効率化・高品位化する」**ために最適化された執筆環境です。
本バージョンでは、内部エンジンを ESM (JavaScript Modules) ベースに刷新し、最新の描画ライブラリを統合しました。

---

## 1. コア・ライティング機能（最新エンジン搭載）

標準的な Markdown に加え、世界標準の最新レンダリングエンジンを搭載しています。

- **標準 Markdown**: 見出し、リスト、テーブル、リンクに加え、**打ち消し線**（`~~`）もサポート。
- **LaTeX 数式 (KaTeX v0.16)**: `$$` で囲むブロック数式や `$ $` のインライン数式を、論文品質で美しく描画。最新の安定版により、描画速度と正確性が向上しています。
- **Mermaid ダイアグラム (v11.4)**: 最新の Mermaid を搭載。XY Chart、ガントチャート、フローチャート、シーケンス図などをテキストから生成。
- **シンタックスハイライト (highlight.js v11.11)**: コードブロックを VS2015 系の落ち着いたダークテーマで強調表示。

---

## 2. 動的コンテンツ＆自動化（効率化の武器）

「一度書いたものを再利用する」「定型作業を自動化する」ための機能群です。

- **Front Matter 変数置換**:
  ドキュメント冒頭の YAML 形式で変数を定義し、本文中の `{{key}}` で呼び出せます。属性、Mermaid、LaTeX 内でも有効です。
- **Macros メニュー**:
  - **日付テンプレート**: 日本語形式、海外形式、ISO形式をワンクリック挿入。
  - **変数・レイアウト雛形**: フロントマターや `<m-d>` タグのサンプルを即座に呼び出し。
- **論理的絶対配置（スタンプ機能）**:
  `::: stamp` を使用して、印影や「社外秘」などの画像を自由な座標に配置可能。

---

## 3. 次世代レイアウト＆アーキテクチャ（v6 の技術的進化）

v6 では保守性と柔軟性を両立する新しい設計思想を導入しました。

- **ESM (JS Modules) アーキテクチャ**:
  プレビューエンジンのライブラリ読み込みを ESM 形式に刷新。最新の JS エコシステムとの高い互換性を確保しました。
- **定数管理システム**:
  Python 側の定数を変更するだけで、Mermaid や KaTeX などのバージョンを一括更新可能なメンテナンス性を実現。
- **魔法のタグ `<m-d>` (Markdown in HTML)**:
  HTML タグの内側で Markdown を記述可能。ネストに応じた自動デデント機能を備えています。
- **コードブロック背景色の動的カスタマイズ (`code_bg`)**:
  フロントマターに `code_bg: "#カラーコード"` と書くだけで、コードブロックの背景色を柔軟に変更。

---

## 4. プロフェッショナル出力 (PDF Fidelity)

「プレビューで見たものが、そのまま PDF になる」ことを徹底追求しています。

- **Fidelity PDF Export**:
  JavaScript による動的なレイアウト計算（改ページ位置など）を物理的に PDF エンジンへ強制継承する独自の同期システム。
- **スマート・ページブレイカー**:
  - **ページガイド表示**: 印刷時の境界線をリアルタイムでプレビュー。
  - **自動余白補正**: 改ページ直後のマージン消失を防止し、WYSIWYG を実現。
- **Refresh Preview (Final Check)**:
  エクスポート直前に最終的なレイアウト整合性を確認するためのリロード機能。
