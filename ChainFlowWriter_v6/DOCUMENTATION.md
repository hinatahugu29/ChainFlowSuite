# ChainFlow Writer v6 総合機能マニュアル

ChainFlow Writerは、美しいドキュメントを素早く書き上げ、PDFとして完成させるための強力な Markdown エディタです。
Mermaid v11 (ESM) および KaTeX v0.16 を搭載し、より高度で安定したドキュメント作成をサポートします。

---

## 1. Markdown 基本・拡張記法
標準的な記法に加え、実用的な拡張機能を搭載しています。

*   **基本装飾**: 太字 (`**`), 斜体 (`*`), 打ち消し (`~~`), インラインコード (`` ` ``)
*   **リスト**: 箇条書き, 番号付きリスト
*   **チェックリスト**: `- [ ]` 未完了, `- [x]` 完了
*   **引用**: `> 引用文` （左側に縦棒が入る標準スタイル）
*   **テーブル**: パイプ記法。右ペインの設定で「枠線あり/なし」を切り替え可能。
*   **脚注**: `[^1]` で注釈を入れ、文末に `[^1]: 解説` で記述。
*   **アンカーリンク**: 見出しに自動的に ID が振られ、文書内リンクが可能です。

## 2. 特別な配置・強調ブロック
報告書やマニュアルで頻出するレイアウトを、独自の記法（:::）で簡単に実現できます。

*   **配置コントロール**:
    *   `::: center` ... `:::`: 中央揃え
    *   `::: right` ... `:::`: 右寄せ
*   **文字サイズ変更**:
    *   `::: large` ... `:::`: 大きな文字（タイトル等に）
    *   `::: small` ... `:::`: 小さな文字（注釈等に）
*   **アドモニション（情報・警告枠）**:
    *   `::: info` ... `:::`: 青色の情報ボックス
    *   `::: warning` ... `:::`: オレンジ色の警告ボックス
*   **論理的絶対配置（スタンプ機能）**:
    *   `::: stamp [CSSプロパティ]` ... `:::`: 文字のフローに追従しつつ、絶対座標で画像を配置。
    *   **質感の再現**: 標準で「乗算(multiply)」がかかり、下の文字と自然に重なります。
    *   **オプション例**: `right:15mm; margin-top:-20mm; transform:rotate(10deg); opacity:0.8;`
    *   **ノーマル指定**: `::: stamp normal;` と書けば透過(乗算)を解除できます。

## 3. 高度なコンテンツ（数式・図解）
Web技術を駆使した最新のレンダリングエンジンを搭載しています。

*   **数式 (LaTeX)**: MathJax による美麗な数式表示。
    *   ブロック数式: `$$ ... $$`
    *   インライン数式: `$ ... $`
*   **図解 (Mermaid)**: テキストで図が描ける Mermaid.js 連携。
    *   ` ```mermaid ` ... ` ``` ` 内に記述。
    *   フローチャート、シーケンス図、ガントチャート等に対応。

## 4. プロパティ・レイアウト設定（右ペイン）
ドキュメントの「見た目」をノンコードで細かく調整できます。

*   **用紙設定**: A4 / B5 / Letter の切り替え。縦(Portrait) / 横(Landscape) の選択。
*   **余白設定**: 上下左右の余白を mm 単位でリアルタイム調整。
    *   **高精度物理スケーリング**: 画面上のピクセルと実寸(mm)を動的に同期。解像度やズーム倍率を問わず、常に100%正確な「紙の大きさ」を再現。
*   **タイポグラフィ**:
    *   フォント切り替え（明朝体 / ゴシック体）
    *   基本文字サイズ (pt) と 行間 (Line Height)
    *   見出しのアンダーライン表示ON/OFF
    *   テーブルのスタイル（枠線あり / 罫線のみのスッキリ表示）
*   **装飾・同期要素**:
    *   ページ番号の自動挿入
    *   全ページ共通のヘッダー（左）とフッター（中央）
    *   **インテリジェント・同期**: 画像や数式など、読み込みに時間がかかるコンテンツがあっても、描画が終わった瞬間にレイアウトを自動再計算して自己修復します。

## 5. PDF 書き出し・印刷制御
*   **PDF出力**: 右ペインの「Export to PDF」から、設定した余白・フォントをそのまま維持して出力。
*   **プロ級のレイアウト整合性**: プレビュー画面の動的な調整機能を、PDF出力時には自動的に印刷エンジン用に最適化。二重余白などのトラブルを防ぎ、美しい一貫性を保ちます。
*   **手動改ページ**: `<div style="page-break-before: always;"></div>` を挿入することで、物理的なスペーサー方式により確実に次のページへとコンテンツを送り出します。
*   **印刷非表示機能**: 
    *   `::: no-print` ... `:::`: この中に書いた内容は**「画面上では見えるが、PDFには出力されない」**ようになります。自分用のメモや未修正箇所の備忘録に最適です。

## 6. チートシート（スニペット管理）
よく使う定型文やHTML構造を効率的に管理できます。

*   **ギャラリー形式**: サムネイル付きで視覚的にスニペットを選択・挿入（Ctrl+H / またはボタン 💡）。
*   **スニペットの登録**: エディタで選択した範囲をそのまま自分専用のスニペットとして登録。
*   **メンテナンス**: 不要になったスニペットは、ギャラリー内で削除可能。画像ファイルも自動で掃除されます。

## 7. 生産性・利便性機能
*   **マルチスレッド描画エンジン**: 
    *   Markdownの解析（レンダリング）をバックグラウンドで実行。
    *   どれほど画像や数式が多くても、エディタのカーソルや文字入力が止まることはありません。
*   **ショートカットキー**:
    *   `Ctrl + S`: 上書き保存
    *   `Ctrl + B`: エディタペイン（左）の表示/非表示を切り替え
    *   `Ctrl + R`: 設定ペイン（右）の表示/非表示を切り替え
    *   `Ctrl + H` (または `Ctrl + Q`): チートシートを開く
*   **ポータビリティ**:
    *   データ（スニペット、サムネイル）はEXEと同じ階層の `snippets_data` フォルダに自動集約。
    *   USBメモリにEXEとフォルダを入れるだけで、どこでも自分の環境が再現されます。
*   **データ確認**: メニュー「表示 > データフォルダを開く」から保存場所を即座に確認。
